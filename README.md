Server Knowledge checklist
============

Provides API to serve user data and knowledge lists.

## Development

To start developing in the project:

1. Install npm modules:
    ```bash
    npm install
    ```
2. Run mongod:
    ```bash
    npm install
    ```

3. Run the server:
    ```bash
    node serve.js
    ```

Then head to `http://localhost:8090` in your browser.
